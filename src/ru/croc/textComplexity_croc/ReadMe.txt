В проект по анализу сложности текста/книги на основе частотности слов входят следующие файлы:
- [ ] Main - содержит метод main и используется для инициализации фабричного метода обработки токенов текста
- [ ] FrequentEnglishWorlds - служит для считывания из файла List'а самых частовстречающихся англ слов
- [ ] FrequencyDictionary  - используется для чтения в словарь пары (токен: частота его встречи) и удаления посторонних символов (==процесс токенизации текста)
- [ ] WordProcessing - тот класс, который имплиментит Runnable для сравнения ключей каждого токена с принадлежностью FrequentEnglishWorlds 

При запуске Main выводятся сообщения пользователю в консоль. При некорректных действиях пользователя предусмотрена обработка большинства ошибок.
В папке Tests, находится тестовые образцы текста (книги, и некоторые лёгкие записки)
